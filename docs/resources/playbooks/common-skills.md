# 常用 Skill 清单

这里的 skill，可以理解成一类高复用的能力模板。它可以是 prompt 模板、工作流套路、文档整理规范，也可以是一个更明确的“任务执行脚本”。

## 我建议优先沉淀的 skill 类型

### 1. 知识整理类

适用场景：

- 看完一篇文章后整理学习笔记
- 把零散资料变成结构化卡片
- 从长文里抽取关键概念和待补问题

可以沉淀的东西：

- 摘要模板
- 知识卡片模板
- 关键词抽取模板

### 2. 写作改写类

适用场景：

- 把口语化内容改成文档
- 统一风格
- 压缩冗长表达
- 展开提纲

### 3. 代码辅助类

适用场景：

- 解释代码
- 生成最小示例
- 查找 bug 风险
- 做 PR review

### 4. 学习复盘类

适用场景：

- 每周总结学习进展
- 找出理解薄弱点
- 规划下周继续补什么

## 一个 skill 应该包含什么

建议至少包含下面四部分：

- 任务目标
- 输入要求
- 输出格式
- 质量标准

如果一个 skill 经常失败，再补：

- 常见误用
- 边界条件
- 失败案例

## 判断一个 skill 值不值得沉淀

如果它满足下面两条以上，就值得写进知识库：

- 一周内会重复用两次以上
- 有明确输入输出格式
- 新人也能复用
- 改得少但收益稳定
