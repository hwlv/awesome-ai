# 资料库

这里不承载主知识主线，而是承载“辅助你持续学习和复用”的资产。

## 资料库里放什么

- 精选参考资料
- 可复用的 playbooks / 提示词模板
- 阅读笔记
- 复盘模板
- 知识模板

## 为什么把 `skills` 放到这里

因为 `skill` 本质上是操作模板，不是 AI 领域本体。

把它和 `agent`、`RAG`、`Prompt` 并列，会把“知识主题”和“执行资产”混在一起。现在它被放到 `resources/playbooks/`，角色会更清楚。
