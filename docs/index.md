## 这个站点适合放什么

- AI 基础概念与术语解释
- 常用 skill、提示词模板、工作流套路
- Agent 设计笔记、模式拆解、踩坑记录
- 参考资料与学习路径
- 可以快速运行和理解的 demo 代码

## 维护原则

1. 文档优先写在 `docs/`，可运行代码优先放在 `examples/`。
2. 每篇文档至少回答四个问题：这是什么、为什么重要、怎么用、容易错在哪。
3. 收藏资料之前先判断是否值得复习，不把这里做成“链接堆”。

## 现在就可以做的事

- 从 [从这里开始](/guide/start-here) 了解整个知识库的使用方式
- 从 [AI 基本概念总览](/fundamentals/ai-basics) 开始搭自己的知识地图
- 从 [示例总览](/examples/) 找一个最小 demo 跑起来
