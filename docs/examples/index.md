# 示例总览

这个栏目对应仓库根目录下的 `examples/`。文档负责解释示例想表达什么，代码目录负责真正承载可运行文件。

## 当前已有示例

### Prompt Basics

- 目录：`examples/prompt-basics`
- 作用：演示一个最小 prompt 结构应该包含哪些部分

### Tool Calling

- 目录：`examples/tool-calling`
- 作用：演示工具注册、工具选择、执行与结果回传的基本循环

### RAG Lite

- 目录：`examples/rag-lite`
- 作用：用一个极简检索脚本解释“先召回，再生成”的思路

### Simple Agent

- 目录：`examples/simple-agent`
- 作用：演示最小 agent loop 如何管理目标、行动和停止条件

## 建议的使用方式

1. 先读对应主题文档。
2. 再跑一个最小示例。
3. 最后自己改一处参数、规则或数据，确认理解是否真正落地。
