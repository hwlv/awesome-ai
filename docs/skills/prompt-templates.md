# 提示词模板

这一页先放最常用、最稳的模板。后面你可以按任务类型不断追加。

## 模板一：知识笔记整理

```text
你是一个技术学习助手。

任务：把输入内容整理为学习笔记。

要求：
1. 先解释核心概念
2. 再列出关键例子
3. 如果有不确定内容，单独列出待确认项
4. 不要编造原文没有的信息

输出格式：
- Summary
- Key Concepts
- Examples
- Open Questions
```

## 模板二：参考资料评估

```text
请评估下面资料是否值得纳入我的 AI 知识库。

评估标准：
1. 是否信息密度高
2. 是否有明确适用场景
3. 是否内容过时风险高
4. 是否值得二刷

输出格式：
- 结论
- 值得看的原因
- 不足
- 推荐放入哪个栏目
```

## 模板三：Agent 工作流拆解

```text
请把这个 agent 方案拆成以下结构：
1. 目标
2. 输入
3. 决策节点
4. 工具使用
5. 状态管理
6. 风险点
7. 最小可行版本
```

## 模板四：每周复盘

```text
请根据下面学习记录，输出本周复盘。

要求：
- 用简洁中文
- 优先总结我真正学会的内容
- 单独列出还模糊的知识点
- 给出下周最值得补的 3 个主题
```

## 使用建议

- 模板先追求稳定，不急着做得很花
- 尽量让输出格式固定，后面更容易沉淀成文档
- 每个模板最好都配一个成功案例和一个失败案例
